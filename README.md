# Projeto_Eletronica_USP
Fonte de tensão desenvolvida na disciplina de Eletrônica (BCC/USP). Converte 127V AC da rede elétrica em 5V DC via transformador, ponte retificadora de diodos, filtragem capacitiva e regulação com LM7805, permitindo o carregamento de dispositivos móveis via USB.
